# RDP(Remote Desktop Protocol)

- RDP(remote desktop protocol) ou (protocolo de área remota): Você tem total controle de um host que tenha o RDP habilitado

   - É como se você estivesse sentado numa frente do pc alvo. você vê tela inteira dele.

- porta padrão: 3389.

- clientes RDP(pra acessar o server RDP)(linux):

   - sintaxe: xfreerdp /v:192.168.0.10 /u:usuario
      - xfreerdp /v:192.168.0.10 /u:usuario /cert:ignore (não verifica o certificado (TLS) de criptografia)
