# Fluxo protocolo

- Protocolo: Conjuntos de regras de rede que definem como PCs devem se comunicas na rede.

   - Como iniciar a comunicação
   - Como enviar dados
   - Como interpretar respostas
   - Como encerrar a conexão

   - Tudo com base em alguma regra específica (SSH,FTP,mysql..)

- Porta: Acesso inicial, onde se conecta para acessar algo(programas,serviços,servidores...)

- Serviços: O que o host está oferecendo na porta:
   - transferências de arquivos sem criptografia(FTP)
   - Acesso remoto seguro(ssh)
   - transferência de hipertextos(informações adicionais de um servidor)(HTTP/HTTPS)
   - Acesso a banco de dados(Mysql)

- Programa: O programa/software que processa e executa dados entre o cliente e o servidor.
    - comandos
    - donwloads,uploads
