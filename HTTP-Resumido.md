# HTTP(Hipertest transfer protocol)

- http(hypertext transfer protocol): Protocolo funciona com o user perguntando algo ao server (request) e o
   server respondendo (reply).
  
- Porta padrão: porta 80

- hipertexto: arquivo que contem links (página html de um site = tem link para as pessoas acessarem)

   - não é cifrado(TLS = top layer securit), que é uma criptografia geral utilizada por serviços web.
      - vulnerável a sniffers de rede.

   - header: cabeçalho, infos extras do server(pode ou não aparecer, depende de como o server foi configurado/protegido)

- como acessar:
   - sintaxe: http://<host/ip>:<porta>
